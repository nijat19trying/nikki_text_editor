# Nikki Text Editor

A simple terminal-based text editor in C for Linux.

## Installation
1. Clone: `git clone https://github.com/nijat19trying/nikki-text-editor.git`
2. Compile: `gcc nikki.c -o nikki`
3. Install: `sudo mv nikki /usr/local/bin/`  # Makes it runnable as `nikki text.txt` (you can use scripts, before using it run "chmod +x 'script_name' and use them like ./script_name"

## Usage
`nikki filename.txt`

Features: Cursor movement, search (Ctrl+F), save (Ctrl+S), quit (Ctrl+Q), delete line (Ctrl+D), etc.
